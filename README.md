## 프로젝트 소개 📝

현재 보유하고 있는 식자재를 기반으로 여러가지 **맞춤 레시피**를 확인할 수 있어요 !

본인의 취향에 맞는 여러가지 레시피 정보를 얻을 수 있어요 !

또한 **먹BTI 조사**를 통해 본인의 식습관과 영양 정보까지 얻을 수 있습니다 🥕
